NetBeansTM IDE

O NetBeans IDE é um ambiente de desenvolvimento integrado (IDE) para escrever, compilar, testar e depurar aplicações desktop e aplicações web usando a plataforma JavaTM. O NetBeans IDE inclui um editor de texto cheio de caracteristicas como sintaxe highlighting e verificação de erros, ferramentas de design visual, suporte ao Ant, suporte a sistemas de controle de versões, e muitos outros recursos.

Para iniciar o IDE (Microsoft Windows) use um destes métodos:

  • Duplo clique no ícone do NetBeans IDE em sua área de trabalho.
  • Escolha Iniciar > Todos os programas > NetBeans 6.0 > NetBeans IDE.
  • Iniciar o IDE em linha de comando C:\> netbeans-install-directory\bin\netbeans.exe.

Para iniciar o IDE (Solaris OS ou Linux):

  • Digite no prompt de comando /netbeans-install-directory/bin/netbeans

Para parar o IDE:

  • A partir do IDE, escolha Arquivo > Sair.

Para desistalar o IDE (Microsoft Windows)

  • Use o utilitário Adicionar ou remover programas no Painel de controle. Não exclua manualmente os diretórios e arquivos.

Para desistalar o IDE (Solaris OS ou Linux)

Use o desistalador fornecido que pode ser encontrado em /netbeans-install-directory/_uninst. Não exclua manualmente os diretórios e arquivos.

Encontrando documentações para o NetBeans IDE

  • Informações sobre os lançamentos do NetBeans IDE 6.0 podem ser encontradas em http://www.netbeans.org/community/releases/60/index.html.
  • O índice da documentação do IDE pode ser encontrado em http://www.netbeans.org/kb/index.html. Este documento também fornece links para mais documentações.
  • A documentação de ajuda está disponível dentro do IDE. A partir do menu, escolha Ajuda > Conteúdo da ajuda. O menu Ajuda também fornece links para outros documentos úteis.

Diretório do usuário do NetBeans IDE

Um diretório de usuário padrão é criado na primeira vez em que o IDE é iniciado. É usado para armazenar suas configurações de usuário, opções e dados de tempo de execução para os servidores pessoais de desenvolvimento. Nos sistemas Microsoft Windows, este diretório é encontrado em \Documents and Settings\user-id\.netbeans\version. No Solaris e no Linux, este diretório é encontrado em /$HOME/netbeans_user
